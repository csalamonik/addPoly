# Function addPoly()

It is function to add two simple polymials represented by two arrays. 

## Inputs/Outputs
    OUTPUT = addPoly(INPUT1, INPUT2)


## Example


#### First polymial  2x^2 + 3 
    INPUT1 = [[2,2], [3,0]]

#### Second polymial 3x^3 + x^2
    INPUT2 = [[3,3], [1,2]]

#### Result 3x^3 + 3x^2 + 3
    OUTPUT = [[3,3], [3,2], [3,0]]
